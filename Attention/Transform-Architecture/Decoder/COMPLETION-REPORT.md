## 🔄 Key Differences: Encoder vs Decoder vs Decoder-Only

| Feature | Encoder | Decoder | Decoder-Only |
|---------|---------|---------|--------------|
| **Self-Attention** | Unmasked (bidirectional) | Masked (causal) | Masked (causal) |
| **Cross-Attention** | None | With Encoder | None |
| **Processing** | Parallel | Sequential | Sequential |
| **Input Constraint** | None | Previous tokens only | Previous tokens only |
| **Typical Use** | Understanding | Generation (seq2seq) | Generation (LLM) |
| **Needs Encoder** | No | Yes | No |
| **Examples** | BERT, RoBERTa | BART, T5 | GPT, LLaMA |
